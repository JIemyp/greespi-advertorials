## **2\. ПОЛІТИКА ТА ОБМЕЖЕННЯ БЕЗПЕКИ (КРИТИЧНО)**

### **A) Правило «Від першої особи» (Meta Anti-Ban)**

Усі тексти: тільки “I / my”.

Без “you/your” у форматі здоров’я або припущення статусу користувача.

✅ SAFE:

* “After my routine change…”  
* “My digestion felt unpredictable…”  
* “When my appetite changed…”  
* “If this season feels familiar…” (замість “If you are in…”)

### **B) Словник (Search vs Social)**

* **SEARCH (Keywords ONLY):** медичні/діагностичні/препаратні терміни дозволені ТІЛЬКИ як ключі в Google Ads (див. Internal Appendix).  
* **SOCIAL/VIDEO:** без назв ліків, без діагнозів, без медичних заяв.

### **C) Product Physics (Frozen Constraints) \- ОСНОВНИЙ ПРОТОКОЛ**

**CRITICAL:** продукт **не можна ділити**. Будь-які інструкції типу “1/2 sachet”, “cut”, “save for later” заборонені.

Позиціонування: **“One gentle portion”** (одна вже м’яка, маленька порція, легка).

**Usage Rule (єдиний допустимий ритуал):**

* **Sachet \+ Water.** \- Без йогуртів/пюре/смузі (додатковий об’єм/смак/калорійність як тригери та зайвий крок).

**Optional Softening (без дозування):**

* “Sip slower” / “Take smaller sips” / “Use more water if I want it lighter.”

### **D) Blacklist (Global Social \+ Visuals)**

* Заборонено: назви діагнозів, назви препаратів, медичні обіцянки/заяви.  
* Заборонено: “treat / heal / fix / reverse / cure / diagnose” (та еквіваленти).  
* Заборонено: “micro-dose” (використовувати “start gentle / start simple / small portion”).  
* Заборонено: “save half for later”, “cut/split”, “defrost before use”.  
* **VISUAL BLACKLIST:** Банки порошков, шейкери, “wellness shakes”, pill-боксы, полки з БАДами. (Замінювати на: блендер, багато посуду, план їжі, велика тарілка, меню).

### **E) Proof Claim Rule**

Будь-які конкретні бейджі (COA/ISO/HACCP/SGS) дозволені **тільки якщо proof публічно клікабельний** на Advertorial/SGS page.

## **5.2 ТЗ: AI-ВІДЕО (TOF)**

### **5.2.1 UGC Creator (Real-person) \- “Calm Adult Confidence”**

* **Архетип:** дорослий, врівноважений, приземлений. Не “інфлюенсер-енергія”, не шоу, не хайп.  
* **Вік-вайб:** 28–45 (може бути ширше, але важливо: *adult, composed*).  
* **Емоційний тон:** спокійна чесність → контроль/полегшення. Без драматизації, без паніки, без “істерики”.  
* **Манера мовлення:** повільніше за середнє, короткі фрази, паузи. Впевненість без агресії.  
* **Обличчя/міміка:** стримана, “я в сезоні обережності, але я зібрався/зібралась”.  
* **Кадр/стиль:** натуральне світло, мінімум фільтрів, чистий фон (кухня/стіл/робочий кут). Ніяких “клінічних” натяків.  
* **Одяг:** нейтральний casual (монохром/спокійні кольори). Без “медичного косплею”.  
* **Поведінка в кадрі:** мінімум жестів, точні дії, “спокійна рутина”.  
* **Заборонено:** комедійні гримаси, перебільшені реакції, “before/after”, “miracle vibe”, “продавлювання” глядача.

### **5.2.2 AI-Video (Generative) \- “Cinematic Realism / Trust-First”**

* **Архетип персонажа:** той самий “calm adult confidence”, але в AI-естетиці (реалістичний актор/актриса або ситуаційний герой).  
* **Візуальна мова:** реалістично, чисто, без гіпер-глянцю. Камера стабільна (без хаотичних зумів/тряски).  
* **Пейсинг:** повільний, читабельний, “quiet confidence”. Кожен кадр має встигнути “прочитатися” без звуку.  
* **Ключовий ефект:** “я впізнаю себе” через ситуацію, а не через продукт.  
* **Деталі, що підсилюють довіру:** прості побутові сцени, мінімалізм, акуратність, відчуття “дорослого рішення”.  
* **Заборонено:** мультиплікаційність, гротеск, надмірні спецефекти, “фантастика”, ультра-глянець, агресивний рекламний тон.

**Нагадування для обох форматів:** TOF \= ідентифікація стану \+ довіра (“gentle \+ proof-first”), без медичних термінів, без обіцянок результату.
