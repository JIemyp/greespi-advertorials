# ЧАСТИНА 1: 8 GOLDEN RULES (Фінальна версія)

## Rule 1: SOCIAL HOOK
**Побутова/соціальна сцена, не симптоми**

```
❌ НЕ: "У мене болів живіт і було здуття"
✅ ТАК: "Вечеря з друзями перетворилася на роботу сапера"
```

**Чому:** Аудиторія втомилася від медичного контенту. Соціальна ситуація створює ідентифікацію без захисних реакцій.

---

## Rule 2: SHRINKING WORLD (з обмеженням)
**Життя звужується через обмеження — БЕЗ хоррору**

```
❌ НЕ: "Залишився рис і страх" (занадто важко)
❌ НЕ: Нагнітання паніки, тривоги
✅ ТАК: "Список скорочувався — залишилось кілька безпечних варіантів"
```

**Тон:** "Впізнавання і видих", не "тиск і паніка"

**Обов'язкові елементи:**
- Динаміка погіршення (не статичний стан)
- Продукти зникають один за одним
- Соціальна ізоляція як наслідок
- ❌ БЕЗ horror-формулювань

---

## Rule 3: TRIED-ENOUGH WALL
**Втома від правил/виключень БЕЗ мед-лексики**

```
❌ НЕ: "Пробіотики, ензими, low-FODMAP, елімінаційні дієти"
✅ ТАК: "Правила, виключення, списки, схеми — я пробувала все"
```

**Заборонено згадувати:**
- Назви дієт (low-FODMAP, GAPS, AIP)
- Назви добавок (пробіотики, ферменти, ензими)
- Медичні терміни

---

## Rule 4: LATERAL DISCOVERY (без claims)
**Дізнався "збоку" — через ПОВЕДІНКОВИЙ контекст, не обіцянку ефекту**

```
❌ НЕ: "Друг використовує для енергії" (= claim)
❌ НЕ: "Це дає апетит" (= причинність)
✅ ТАК: "Друг згадав у контексті 'днів, коли не хочеться готувати'"
✅ ТАК: "Коли потрібна проста база без зайвих рішень"
```

**Безпечні поведінкові контексти:**
- "Дні, коли не хочеться готувати"
- "Коли потрібна проста база"
- "Менше рішень вранці"
- "Щось стабільне без думання"

**Чому зміна:** "Для енергії/апетиту" = причинність = claim. Поведінковий контекст безпечний.

---

## Rule 5: PHYSICS AS TRUST
**Frozen/cold chain/unboxing як інфраструктура довіри**

```
Обов'язкова сцена UNBOXING:
- Термобокс
- Холодна доставка
- "Це їжа, не порошок"
```

**Чому працює:**
- Виправдовує ціну
- Відрізняє від БАДів
- Фізичний доказ > слова

---

## Rule 6: VERIFIED DETAILS ONLY ⚠️ NEW
**Конкретика ТІЛЬКИ якщо підтверджена офіційно**

```
ПРАВИЛО:
Деталі типу "60 секунд", "гладка текстура", "твердий як камінь"
допускаються ТІЛЬКИ якщо підтверджені Start Guide або офіційним описом.

Якщо НЕ підтверджено → нейтральні формулювання:
```

| Якщо не підтверджено | Використовуй замість |
|---------------------|---------------------|
| "60 секунд" | "простий ритуал" |
| "гладка текстура" | "приємна консистенція" |
| "твердий як камінь" | "заморожений, холодний" |
| "без пісочності" | "реальна їжа" |

**Чому:** Невірна деталь = trust risk + потенційний chargeback/compliance ризик.

---

## Rule 7: RETENTION LOOP (через ритуал, БЕЗ доз)
**Зрив звички → повернення шуму → відновлення ритуалу**

```
❌ НЕ: "Перейшов на 1 саше замість 2 → провал енергії"
✅ ТАК: "Пропустив ритуал через поїздку → шум повернувся → відновив звичку"
```

**Механіка:**
1. Герой відчуває стабільність
2. Життєва ситуація зриває ритуал (поїздка, завал, забув)
3. Повертається "шум" / втрата контролю
4. Повертає ритуал → знову тиша

---

## Rule 8: GENTLE CTA
**"Подивися формат/набори/ціну" без тиску**

```
❌ НЕ: "Купи зараз, щоб вилікуватися!"
❌ НЕ: "Тільки сьогодні знижка!"
✅ ТАК: "Якщо сезон знайомий — подивися, що там є"
```

**Елементи soft CTA:**
- Пояснити, що побачить на сторінці
- Без urgency/scarcity
- "Подивися" замість "Купи"

# ЧАСТИНА 3: COMPLIANCE CHECKLIST v1.2

## ✅ MUST HAVE (Обов'язково):

### Формат і структура:
- [ ] Тільки перша особа ("я", "мене", "мій")
- [ ] **DISCLOSURE** на початку або в кінці тексту
- [ ] 16-Beat структура дотримана

### Обов'язкові елементи:
- [ ] Social Hook (побутова сцена, не симптоми)
- [ ] Shrinking List (динаміка звуження)
- [ ] Lateral Discovery (друг з ІНШОГО контексту)
- [ ] Cold Chain / Unboxing сцена
- [ ] Quiet Result ("тиша", не "вилікувався")
- [ ] Retention Loop через РИТУАЛ (не дозу)
- [ ] Soft CTA без тиску
- [ ] Return-to-Social як фінальна нагорода

### Перевірка конкретики:
- [ ] Всі деталі (час, текстура, температура) **підтверджені Start Guide**
- [ ] Якщо не підтверджено → нейтральні формулювання

---

## ❌ MUST NOT HAVE (Заборонено):

### Таймлайни та дози:
- [ ] ❌ Day N ("день 5", "день 20", "через 4 дні")
- [ ] ❌ Конкретні терміни ("2+ місяці", "через 3 тижні")
- [ ] ❌ Дозування ("1 саше vs 2 саше", "зменшив порцію")

### Медична лексика:
- [ ] ❌ Назви дієт: low-FODMAP, елімінаційна, кето
- [ ] ❌ Назви добавок: пробіотики, ензими, ферменти
- [ ] ❌ Вітаміни, мінерали, нутрієнти з дозами
- [ ] ❌ Слово "спіруліна"
- [ ] ❌ Медичні діагнози (IBS, SIBO, leaky gut)

### Claims:
- [ ] ❌ "Для енергії" / "дає апетит" (= причинність)
- [ ] ❌ "Вилікував", "пройшло", "зникли симптоми"
- [ ] ❌ Обіцянки результату

### Тон:
- [ ] ❌ Horror-формулювання ("рис і страх")
- [ ] ❌ Нагнітання паніки/тривоги
- [ ] ❌ Занадто важкий емоційний тиск

### Продажний тиск:
- [ ] ❌ Urgency: "тільки сьогодні", "останній шанс"
- [ ] ❌ Scarcity: "залишилось 5 шт", "розпродано"
- [ ] ❌ Порівняння з конкурентами

### Непідтверджена конкретика:
- [ ] ❌ "60 секунд" (якщо не в Start Guide)
- [ ] ❌ "Гладка текстура" (якщо не підтверджено)
- [ ] ❌ Будь-які деталі без офіційного джерела

# ЧАСТИНА 4: DISCLOSURE FORMATS

## Обов'язкова маркіровка (обрати один варіант):

### Варіант А: На початку тексту
```
*(Особистий досвід. Не є медичною порадою)*
```

### Варіант Б: Партнерський матеріал
```
*Партнерський матеріал*
```

### Варіант В: Sponsored (для Meta/Google)
```
*Sponsored content*
```

### Варіант Г: Розгорнутий disclaimer (в кінці)
```
*Це особистий досвід автора. Результати індивідуальні. 
Greespi — функціональна їжа, не лікарський засіб. 
Перед змінами в раціоні проконсультуйтесь з лікарем.*
```

**Правило:** Формат disclosure залежить від платформи. Перевірити вимоги площадки ПЕРЕД публікацією.
